## Periodic Tasks (every 60 minutes)

### Priority Checks
1. Check for urgent messages or mentions requiring response
2. Review any pending approvals or decisions
3. Check for new tasks or assignments

### Maintenance
1. Review recent memory files for important updates
2. Check workspace for any files needing organization
3. Verify any ongoing tasks status

### Notes
- Only alert if something requires immediate attention
- Otherwise respond with HEARTBEAT_OK
